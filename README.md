# 🌤️ Weather App

A simple app to check current weather using OpenWeatherMap API.

## 🔧 Built With
- HTML, CSS, JavaScript
- Bootstrap
- [OpenWeatherMap API](https://openweathermap.org/api)

## 🚀 How to Use
1. Enter a city name (e.g., London)
2. Click "Get Weather"
3. See the current temperature and conditions
